Movie Mania

Developed as part of a capstone course at Rowan University, the MovieMania Project is a web application that enables users to create a personal library for adding movies to a "Watch Later" list. It also allows users to catalog and store watched movies for future reference and provides movie recommendations based on their "Watch Later" list. The project is built using HTML, JavaScript, and JSONbin.io, with the TMDB API utilized for fetching movie details and recommendations. As the backend developer, I was responsible for implementing data storage and API integration. The project was a collaborative effort, with contributions from David Ranoia, Eric Robinson, Haley Marvasi, and Nick Ciquero.
